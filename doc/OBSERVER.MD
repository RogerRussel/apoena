# Observer Methods

## run
Method run, exec all eyes setted into conf

<code>
$observer->run();
</code>

## runnit
method runnit, exec an especific eye setted at conf

<code>
$observer->runnit('nfs');
</code>

## writeJson
Set http header to Json Content-Type and write result as json.

## getResult
get the result from monitoring as an array.
